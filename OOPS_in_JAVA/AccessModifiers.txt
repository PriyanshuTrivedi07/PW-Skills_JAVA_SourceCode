Defaul value :
    if we do not initialize the values after declaring an object , then all properties have some default values .
    e.g. ->  int have default value '0', String have default value "null" , double have "0.0" etc. 

Access Modifiers:
    (1) Public - all packages
    (2) Private - same class
    (3) Default - same package

Getters and Setters :
